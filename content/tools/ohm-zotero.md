The OHM data pipeline is rooted in Zotero: each source — a paper, a map,
a chronicle, a dataset — lives as a Zotero item, and OHM-specific
metadata is carried as `ohm:*` tags on that item.

This plugin adds dedicated UI for editing those descriptors, so curators
can work directly in Zotero rather than maintaining a parallel spreadsheet.
Pair it with the Chrome extension below to capture sources from the web.
